# Databricks Bingo

A team training game that turns the Databricks platform into an interactive Bingo experience. A Bingo Master reads scenario questions aloud; players identify the Databricks feature that answers each question and mark it on their card.

## Play Now

**https://jamesl-databricks.github.io/databricks-bingo/**

- **Bingo Master** — click "I'm the Bingo Master" to run the game
- **Players** — click "I'm a Player" to get your own randomized bingo card

Each player gets a unique 5x5 board every time they load the page. No accounts, no setup — just open the link and play.

## How to Play

1. One person opens the link and clicks **"I'm the Bingo Master"**
2. Each player opens the link on their own device and clicks **"I'm a Player"** — they get a unique randomized card
3. The Bingo Master reads each scenario question aloud to the group
4. After discussion, the Master clicks **"Reveal Answer"** to show the Databricks feature
5. Players check if that feature is on their card and tap to mark it
6. First player to complete **5 in a row** (horizontal, vertical, or diagonal) calls **"BINGO!"**
7. The Master uses the **Verify Bingo** panel to confirm the claim is valid

## Features

- **45+ scenario questions** mapped to unique Databricks platform features
- **Randomized cards** — each player gets a different 5x5 layout (24 features + FREE center)
- **Bingo detection** — automatically checks all rows, columns, and diagonals
- **Verification tool** — Master can confirm claimed bingo squares were actually called
- **Confetti animation** on valid bingo
- **Fully responsive** — works on phones, tablets, and desktops
- **Zero dependencies** — no build step, no backend, no network required after page load
